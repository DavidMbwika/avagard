WordPress Plugin

Setup Guide

  1. Download and install the Clix plugin.
  2. Activate the plugin and enter your API key.
  3. Use the shortcode [clix-stk-push] to add a payment button.

HTML & JavaScript

For basic websites. Just copy-paste the code below.

Step 1: Add this button code

<button onclick="sendSTKPush()">Pay with M-Pesa</button>

Step 2: Add this JavaScript code

function sendSTKPush() {
    fetch('https://api.clix.co.ke/payments/stk-push', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            phone: "+254712345678",
            amount: 100,
            businessID: "YOUR_BUSINESS_ID"
        })
    }).then(response => response.json())
      .then(data => console.log(data))
      .catch(error => console.error(error));
}

FAQs

How long does an STK push take?

Payments are processed instantly, usually within 10-15 seconds.

What happens if a payment fails?

Failed payments are logged, and you can retry them from your dashboard.

Can I use this on multiple websites?
